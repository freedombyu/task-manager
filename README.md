# Interactive Task Manager

A modern task management application demonstrating JavaScript and React concepts.

## Features

- Manage tasks interactively
- Built with React and ES6
- Responsive design

## Installation

npm install

## Usage

npm start

## Scripts

- start: Launch the development server
- build: Create a production build
- test: Run tests
- eject: Eject configuration

## Dependencies

- lucide-react
- react
- react-dom
